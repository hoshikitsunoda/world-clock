# world-clock
A website for people who want to know times in different regions.
# Usage
```bash
$ node index.js
```
# Uses
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* Moment
* Moment Timezone
* MongoDB
* Express
* Node.js
